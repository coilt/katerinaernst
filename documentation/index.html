1.1 Requirements

There are system requirements in order to install and setup Most template and its components properly. Make sure that you are running the node.js and npm in your computer. You have to maintain this version (Node js >= 14.0.0 and npm >= 5.6)

Recommended Version Nodejs v16.13.2

If you don't have Node js in your computer please download and install from here: https://nodejs.org/en/download/

Install Packages

You can use Npm/Yarn

NPM : You must have to npm install packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

								
									npm install

Yarn: At first install yarn globally by npm install --global yarn. You must have to yarn install packages. You can do this by running yarn install from the root of your project to install all the necessary dependencies.

								
									yarn install
Development server

Run npm run dev for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files.

								
									npm run dev / yarn dev
Build

Run npm run build to build the project. The build artifacts will be stored in the .build/ directory.

								
									npm run build / yarn build

Run npm run start to start build version in your localhost.

								
									npm run dev / yarn dev
